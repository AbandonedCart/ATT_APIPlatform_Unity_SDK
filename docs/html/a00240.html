GetMessageDeltaResponse ATT_MSSDK.RequestFactory.GetMessageDelta ( string  state)

This method provides capability to check for updates by passing in a client state. This is typically used when a client goes from being offline to becoming online. If the subscriber’s mailbox index cache does not exist an error (MESSAGES.MESSAGES-4019) would be returned to the client and the client would have to re-initialize the cache by calling CreateMessageIndex method.

Parameters:
stateThe state of the client which was obtained from a either the Get Message Index request, or from the Get Message List request.
Returns:
GetMessageDeltaResponse object.

NotificationConnectionResponse ATT_MSSDK.RequestFactory.GetNotificationConnectionDetails ( string  queues)

This method provides capability to retrieve details about the credentials, endpoint and resource information to setup a notification connection. This method needs to be invoked before a STOMP a connection is established to subscribe for notifications. The user credentials provided in the response is valid for a period of 30 seconds and STOMP connection needs to be established within this period. If the credentials expire, this method would needs to be invoked again.

Parameters:
queuesThe name of the resource the client is interested in subscribing for notifications. Currently supported resource is TEXT – Subscription to this resource will provide notification related to messages stored as TEXT in the cloud inbox. MMS – Subscription to this resource will provide notification related to messages stored as MMS in the cloud inbox.
Returns:
An instance of NotificationConnectionResponse class.
