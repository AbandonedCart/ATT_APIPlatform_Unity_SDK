AT&T API Platform APIs
SMS
The Short Message Service (SMS) API enables your application to reliably send and receive secure text messages to mobile devices on the AT&T network.
  • Send SMS
  • Get SMS Status
  • Receive SMS
  • Get SMS
  • Receive SMS Delivery Status
MMS
The Multimedia Messaging Service (MMS) API enables your application to reliably send and receive messages that include video, images, and audio, in addition to text, and get the delivery status of sent messages.
  • Send MMS
  • Get MMS Status
  • Receive MMS
  • Receive MMS Delivery Status
Device Capabilities
The Device Capabilities API enables your application to identify the capabilities of a mobile device on the AT&T network.
  • Get Device Capabilities

Device Location
The Location API enables your application to determine the location of a mobile device on the AT&T network.

  • Get Device Location

In-application Messaging from Mobile Number
The In-application Messaging from Mobile Number (IMMN) API enables your application to send and receive SMS and MMS messages on the behalf of mobile devices for the user.

  • Send Message
  • Get Message Headers
  • Get Message Content

Call Management (Beta)
The Call Management (Beta) API, powered by Tropo®, enables your application to recognize voice commands that send or receive calls, provide Interactive Voice Response (IVR) logic, Automatic Speech Recognition (ASR), Voice to Text (VTT), Text (SMS) integration, and more. If an application hosts a service that complies with an AT&T specified callback, the application can also receive MMS messages.

  • Create Session
  • Send Signal

Payment
The Payment API enables your application to charge digital services directly to an AT&T subscriber's bill for payment.

  • New Transaction
  • Get Transaction status
  • Transaction Refund
  • Transaction Notification
  • Create Subscription
  • Get Subscription Status
  • Get Subscription Details
  • Subscription Refund
  • Subscription Notification

Notary
The Notary API enables your application to digitally sign content so that recipients can trust that the content is unaltered and from the person who sent it.
  • Sign Payload
Speech
The Speech API enables apps to convert speech (audio) to text using different contexts.
  • Speech To Text
Text To Speech
The Text to Speech API enables your application to convert text to different speech audio formats.
  • Text to Speech
Speech To Text Custom
The Speech To Text Custom API enables your applications to convert speech to text using custom inline grammars or hints.
  • Speech To Text Custom
ADS
The Advertisement API enables your application to support advertisements within the app. The application developer can collect a revenue share of the advertisement. When users click the advertisement in the app, they are redirected to the web page for the advertisement.
  • Get Ads
